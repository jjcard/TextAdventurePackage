TextAdventurePackage
====================

A java library to help make text adventure games.
Includes interfaces and basic implimentation classes for:

  - Location
  - Mob
  - Player
  - Exit
  - Item
  - Weapon
  - Armour
  - Text Parser
  - Game Loop
